## アミノ酸代謝の異常について

**フェニルケトン**尿症, **メープルシロップ**尿症, **ホモシスチン**
尿症の全て**新生児マススクリーニング**の対象である。

### フェニルケトン尿症

|      |                                |
| ---- | ------------------------------ |
| 欠損 | **フェニルアラニン水酸化酵素** |
| 異常 | **フェニルアラニン**⬆️         |
|      | **チロシン**⬇️                 |

- **ネズミ臭**: **フェニルアラニン**は **フェニルピルビン酸**
  になって尿中に排出される。
- **中枢神経症状**& **神経発達遅滞**: **血液脳関門**の
  **大型中性輸送体**を占有して他の **大型中性アミノ酸**を欠乏させる。
- **色白**: **チロシン不足**で、誘導体である **メラニン**が不足する。
- **神経伝達物質の不足**: **チロシン不足**で、 **アドレナリン**,
  **ノルアドレナリン**, **ドーパミン**が不足する。

### メープルシロップ尿症

|      |                                   |
| ---- | --------------------------------- |
| 欠損 | **分枝 α-ケト酸デヒドロゲナーゼ** |
| 異常 | **α-ケト酸**⬆️                    |

**ロイシン**, **バリン**, **イソロイシン**の **アミノ基転移**反応後の
**酸化的脱炭酸**反応ができない。

- **脳毒性**

|      |                               |

| 異常 | **メチオニン**⬆️              |
|      | **ホモシスチン**⬆️            |

- **知能障害**
- **水晶体脱臼**

## 先天性副腎皮質過形成

定義的には **ステロイドホルモンの酵素**欠損による疾患の総称。
最も多いのは **21-水酸化酵素欠損症**。

|      |                            |
| ---- | -------------------------- |
| 欠損 | **21-水酸化酵素欠損症**    |
| 症状 | **副腎皮質刺激ホルモン**⬆️ |
|      | **コルチゾール**⬇️         |
|      | **アルドステロン**⬇️       |

- **低血糖**, **低血圧**, **塩喪失**: **コルチゾール**と
  **アルドステロン**不足。
- **女性外性器の男性化**: **アンドロゲン**の過剰。
